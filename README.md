# HabitSehat

Aplikasi Android utility untuk membangun kebiasaan sehat dan melacak konsumsi air minum.

## Fitur

- **Habit Tracker** — Catat dan lacak kebiasaan harian (olahraga, baca, meditasi, dll)
- **Water Intake Tracker** — Pantau asupan air minum dengan pengingat
- **Streak & Heatmap** — Lihat konsistensi dengan kalender seperti GitHub contribution graph
- **Material Design 3** — UI modern dengan Material You, mendukung dark mode
- **Fully Offline** — Semua data disimpan lokal, tanpa perlu akun
- **Ringan** — Target APK < 5MB
- **Widget Home Screen** — Lihat progress tanpa buka aplikasi (coming soon)

## Tech Stack

- **Kotlin** + **Jetpack Compose** — Modern declarative UI
- **Room Database** — Local storage
- **DataStore Preferences** — User settings
- **WorkManager** — Background reminder
- **Material Design 3** — Design system

## Minimum Requirements

- Android 8.0+ (API 26)
- 1GB RAM
- ~5MB storage

## Lisensi

Hak cipta dilindungi. Aplikasi ini adalah produk komersial.
