Kebijakan Privasi HabitSehat

Terakhir diperbarui: 14 Juli 2026 | Versi 1.0

Ringkasan Singkat: HabitSehat adalah aplikasi 100% offline. Kami tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda ke server manapun. Semua data tersimpan lokal di perangkat Anda.

1. Data Yang Kami Kumpulkan

Jawaban singkat: Tidak ada.

HabitSehat tidak mengumpulkan data pribadi, data kebiasaan, data kesehatan, lokasi, kontak, atau identitas perangkat. Aplikasi berfungsi sepenuhnya tanpa koneksi internet.

2. Data Yang Tersimpan Lokal (Di Perangkat Anda)

Data berikut hanya tersimpan di database lokal (Room/SQLite) pada perangkat Anda:

  • Daftar kebiasaan (nama, ikon, kategori, target, jadwal)
  • Log harian kebiasaan (tanggal, jumlah, catatan)
  • Log minum air (tanggal, jumlah ml, jenis minuman)
  • Kebiasaan buruk (HabitStop): nama, biaya per kejadian, log resistensi
  • Sesi Pomodoro (durasi, tipe, habit terkait, tanggal)
  • Progress challenge & badge yang didapat
  • Penggunaan: tema terpilih, mode gelap/terang, target air harian
  • Status premium (lokal, divalidasi via Play Billing)

Data ini tidak pernah keluar dari perangkat Anda kecuali Anda memilih fitur Backup ke Google Drive (hanya premium, manual, terenkripsi).

3. Izin Aplikasi

IzinAlasan
POST_NOTIFICATIONSPengingat minum air, notifikasi laporan mingguan, Pomodoro selesai
RECEIVE_BOOT_COMPLETEDJadwal ulang notifikasi & widget update setelah reboot

Kami tidak meminta izin: Lokasi, Kamera, Mikrofon, Kontak, SMS, Telepon, Penyimpanan (kecuali scoped storage untuk backup file).

4. Layanan Pihak Ketiga

  • Google Play Billing — Validasi pembelian premium (langsung via Play Store, kami tidak melihat detail pembayaran)
  • Google Drive API — Hanya jika Anda manual klik "Backup ke Drive" (premium). Data dienkripsi sebelum upload.
  • Firebase/Analytics/Crashlytics — TIDAK DIGUNAKAN di aplikasi ini.

5. Keamanan Data

  • Database lokal: Room (SQLite) dengan enkripsi filesystem Android standar
  • Backup Google Drive: File JSON dienkripsi AES-256 sebelum upload
  • Tidak ada komunikasi jaringan latar belakang (background network)

6. Hak Pengguna

Anda memiliki kontrol penuh atas data Anda:

  • Hapus semua data: Menu Pengaturan → "Hapus Semua Data" (sekali klik, permanen)
  • Ekspor data: Fitur backup manual ke Google Drive (premium)
  • Uninstall: Menghapus aplikasi = menghapus semua data lokal

7. Anak di Bawah 13 Tahun

HabitSehat tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak sadar mengumpulkan data dari anak-anak. Jika Anda orang tua dan menemukan anak Anda menggunakan aplikasi ini, tidak ada risiko privasi karena tidak ada data yang dikumpulkan.

8. Perubahan Kebijakan

Jika ada perubahan materiil (misal: menambahkan fitur sync cloud opt-in), kami akan memperbarui halaman ini dan menampilkan notifikasi di aplikasi. Versi terbaru selalu tersedia di tautan ini.
